PROJECT: hijabbyvla (PHP backend + Admin + static frontend)

How to use:
1. Extract this folder into your webserver directory (e.g., XAMPP htdocs/hijabbyvla or /var/www/html/hijabbyvla).
2. Ensure folder 'uploads/' is writable by webserver.
3. Import 'init.sql' into phpMyAdmin to create database and tables.
4. Edit config.php if your DB credentials are different.
5. Open admin panel: http://localhost/hijabbyvla/admin/index.html
6. Open frontend: http://localhost/hijabbyvla/public/index.html

Notes:
- Replace the example admin password hash in init.sql with a real hash generated by PHP's password_hash().
- This project is for development/testing. For production, add authentication, CSRF protection, input validation, and secure CORS settings.
